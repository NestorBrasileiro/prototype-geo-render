Sobre

GeoMVT é um visualizador de alto desempenho para Mapbox Vector Tiles gerados dinamicamente a partir de dados geoespaciais armazenados no MongoDB. Os pontos são servidos via protocolo MVT (protobuf) e renderizados diretamente no navegador com MapLibre GL JS.
